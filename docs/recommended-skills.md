# 推荐 Skills

本项目已安装以下 AI Skills，安装后会自动加载详细规则，此处仅作速查索引。

---

## 1. remotion-best-practices

**来源**: [remotion-dev/remotion/packages/skills](https://github.com/remotion-dev/remotion/tree/main/packages/skills)

Remotion 视频开发最佳实践知识库，涵盖 3D 场景、动画、字幕、音频、转场、图表等 28 条规则。处理任何 Remotion 代码时自动加载。

---

## 2. aetherviz-master

**来源**: [andyhuo520/aetherviz-master](https://github.com/andyhuo520/aetherviz-master)

互动教育可视化建筑师，输入教学主题即可生成沉浸式 3D 交互教学网页。可用于快速验证教学逻辑与交互效果，再迁移到 Remotion 视频中。
